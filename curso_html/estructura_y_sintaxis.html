Estructura y sintaxis del lenguaje HTML

El lenguaje HTML (Hypertext Markup Language) es un lenguaje de marcado utilizado para estructurar el contenido de las páginas web y definir su apariencia.

La sintaxis de HTML es relativamente sencilla y consta de una serie de etiquetas que se utilizan para indicar cómo se debe mostrar el contenido en la pantalla del usuario.

Las etiquetas de HTML permiten definir títulos, párrafos, enlaces, imágenes y otros elementos que se encuentran en una página web.

Al utilizar estas etiquetas de forma adecuada, podemos estructurar el contenido de una página de manera clara y organizada, lo que facilita la lectura y comprensión por parte del usuario.

Etiquetas

La estructura fundamental de una etiqueta es la siguiente:

<etiqueta> </etiqueta>

Como vemos, la sintaxis consta de dos partes, una primera etiqueta de apertura con un nombre dentro encerrado entre los símbolos < y >.

Y una segunda etiqueta de cierre, con el mismo nombre de la etiqueta de apertura, precedida de </ y cerrada con >.

En ocasiones, usaremos etiquetas sin contenido y podremos cerrarla en la misma etiqueta de apertura.

<etiqueta-sin-cierre/>

Dentro de la etiqueta de apertura podremos usar determinados atributos. Los atributos serán propiedades de las etiquetas y contendrán un valor.

<etiqueta atributo1="valor1" atributo2="valor2">
